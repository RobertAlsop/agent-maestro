# Exploration Pipeline

## Purpose

Discover opportunities for operational intelligence within an organization or domain.

## When It Is Used

At the start of any new engagement, project, or capability area. Corresponds to the Exploration [[Agent Maestro — Modes|Mode]].

## Stages

1. Map the target operational domain
2. Identify high-value workflow candidates for agent augmentation
3. Audit existing tools, data, and integration points
4. Assess governance readiness
5. Define candidate use cases with rough scope

## Inputs

- Organizational context
- Workflow descriptions
- Access to operational stakeholders

## Outputs

- Opportunity map
- Candidate use case shortlist
- Governance readiness assessment
