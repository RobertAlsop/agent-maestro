# Organizational Intelligence

## Purpose

The domain of how organisations develop, store, and use intelligence — both human and machine — to improve decisions, operations, and adaptive capacity over time. Organisational Intelligence extends classical knowledge management into the era of AI-native organisations.

## Context

In Agent Maestro's frame, Organisational Intelligence is the domain in which agent-generated [[Operational Intelligence]] accumulates into organisational knowledge. It covers how insights from agent operations become organisational assets — through memory architecture, operational feedback loops, and governance-compliant learning.
