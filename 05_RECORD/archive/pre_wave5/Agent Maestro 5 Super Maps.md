# Why This Works at Large Scale

When the vault grows to thousands of notes:

Without maps:

- graph becomes chaotic
    
- duplication appears
    
- navigation slows down
    

With super maps:

- every note has a clear anchor
    
- conceptual drift is reduced
    
- the graph becomes navigable

# The One Rule That Makes This Work

Every note should link to **at least one super map**.

Preferably:

1 concept map  
1 system map  
1 pipeline or project map

That simple rule keeps the graph stable even as it grows.
